A tag p é para parágrafos;

A tag mark é para destacar semânticamente um trecho de um texto em seu contexto

A tag small é para deixar um texto menor tipo assim Hello

A tag i deixa o texto em itálico como esse: Hello

A tag u sublinha o texto

A tag strong dá uma importância semântica ao seu texto indicando como assunto principal do site

A tag ol cria listas ordenadas como essa:

  1. item 1
  2. item 2
  3. item 3

A tag ul cria listas não ordenadas como essa:

  • item 1
  • item 2
  • item 3

A tag a cria links para páginas externas ou até mesmo para partes da mesma página como esses:

Início

Google

A tag hr cria uma linha horizontal na página, como essas várias vistas nessa página,

A tag sub é utilizada para escrever um texto com letras menores e com o alinhamento abaixo do resto do conteúdo como essa Hello

Já a sup é utilizada para escrever um texto com letras menores e com o alinhamento acima do resto do conteúdo como essa Hello

A tag blockquote define uma seção [dentro de um documento] que é citada de outra fonte.

A tag del (ou Elemento HTML de Texto Excluído) representa uma parte do texto que foi excluída de um documento. Este elemento é (não necessariamente) renderizado pelos navegadores com uma linha entre o texto.

A tag abbr é para abreviações, o atributo title é usado para descrever a abreviação.
